How to Edit User Profile Details

How to

In this screen customer can see Three panel first panel for Personal detail, second panel for Account deatil and third panel for Security Question. Customer can edit only his Personal detail to Click on Edit button.

If the user Click on button. User can edit his Personal Detail (Like Primary Number, Email and Address detail etc). User also update his Profile picture to this section. After editing the Personal detail user can save his detail to click the Save button.
