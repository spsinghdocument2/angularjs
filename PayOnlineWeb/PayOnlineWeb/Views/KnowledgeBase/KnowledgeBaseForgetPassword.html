What if You Forgot Password

How to

In case of Forgot Password, user can reset the password from login screen and to do so you have to verify the MAF Account Details and Security Details.

To retrieve the password, go to Login page which is the default page, and Click on Forgot Password link as shown in image.

Then, you will be redirected to Step 1 for account verification.

Step 1 : Enter MAF Account Number

In this step, enter your MAF Account Number from the file/account details. Then Verify the account details by Clicking on button and if information is verifed successfully then you will be moved to Step 2.

Step 2 : Verfiy the security question
Case 1. If you know the secuity question.

In this step, you have to verfiy the "Security Question" with "Answer" which is associated with the information from first step and Click on button.

Once the entered information will be validated, the password will be changes with temprory password and an email confirmation is send to the customer with login information.

Case 2. If you don’t know the security question

If you don’t know the security question then Click on “If you don’t have any security question” link and a popup window will apear with the information to contact customer service.

User can return to "MAF Login" page anytime from the forgot password process, and to do so, have to Click on Return to "MAF Login" page.
